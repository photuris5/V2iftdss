BehavePlus SCORCH Compared to IFT-scorch

IFT-scorch calculates scorch height using manually entered variables for flame length, midflame wind speed, and air temperature. The option to calculate crown scorch using fireline intensity has not been implemented in IFTDSS, nor has the option to link IFTDSS fire behavior modules to the crown scorch module for automatically populating the flame length input or fireline intensity.
