IFTDSS Compared with Other Systems

The IFTDSS software integration framework provides a portal to the scientific algorithms and models that exist in commonly used fire and fuels software applications. The modules in IFTDSS are organized to fulfill specific business needs, which have been defined as IFTDSS workflows. In some cases, only specific functional pieces of larger applications have been implemented. Many modules and software applications have been and will be added to IFTDSS.

Quality assurance tasks are performed to ensure the modules implemented in IFTDSS provide the same numerical values as the standalone desktop versions of the modules. Additionally, the functionality of the modules implemented in IFTDSS has been compared with the original standalone desktop applications. These comparisons were made to inform users about the functionality that is and is not included in the IFTDSS implementation.

To obtain module-specific quality assurance test information, see Module Test Cases.

To date, the following comparisons have been conducted:

  • A Comparison of the Functionality in BehavePlus 5.0.5 and in IFTDSS Version 1.0 (a technical memorandum in PDF format)
  • IFTDSS Surface Fire Behavior Output Evaluation (a technical memorandum in PDF format)
  • Numerical comparison of IFTDSS, FlamMap, BehavePlus, and NEXUS fire behavior models; Appendix 1 and Appendix 2 provide tables of the numerical comparisons.
