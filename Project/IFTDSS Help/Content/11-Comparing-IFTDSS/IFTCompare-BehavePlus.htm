BehavePlus Functionality Available in IFTDSS Version 1.0

In IFTDSS, eight of the nine software modules (available in BehavePlus 5.0.5) for simulating stand level fire behavior potentials—CONTAIN, CROWN, IGNITE, SAFETY, SCORCH, SIZE, SPOT, and SURFACE —have been implemented using the Fire Behavior Software Development Kit (FBSDK) code (http://www.fire.org) and BehavePlus code libraries provided by Systems for Environmental Management (SEM).

The ninth software module available in BehavePlus, MORTALITY (tree mortality), has not been implemented in IFTDSS; instead, tree mortality in IFTDSS can be calculated using the latest equations available in the First Order Fire Effects Model (FOFEM).

BehavePlus 5.0.5 provides six tools for estimating additional variables, such as fine fuel moisture or slope; one of these tools, the unit converter tool, is available in IFTDSS as the Unit Converter. The other five tools (fine dead fuel moisture, relative humidity, slope from map measurements, and the slope verses horizontal map distance tools) are not available in IFTDSS, but are being considered for inclusion in future versions on the basis of end-user needs and feedback. There are currently no plans to include the Sun-Moon calendar in IFTDSS.
