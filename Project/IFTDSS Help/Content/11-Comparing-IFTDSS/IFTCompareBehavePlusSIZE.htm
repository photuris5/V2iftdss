BehavePlus SIZE Compared to IFT-size

IFT-size includes all the BehavePlus SIZE input and output options except for the fire shape diagram.

IFTDSS also includes linking and automatically populating the appropriate size estimation module inputs with the appropriate surface fire behavior outputs.
