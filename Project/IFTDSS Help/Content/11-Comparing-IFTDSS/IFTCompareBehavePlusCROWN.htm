BehavePlus CROWN Compared to IFT-crown

Input Parameters

Through IFT-crown, IFTDSS provides 13 of the 14 BehavePlus CROWN module input parameter options.

The one exception is that in BehavePlus, the user is provided the option of representing surface fire intensity using either flame length or fire line intensity (which is calculated from flame length). In IFTDSS, the user is required to enter flame length. In IFTDSS, the user needs to run the surface fire behavior module first and then manually input the surface module outputs into the crown fire behavior module.
