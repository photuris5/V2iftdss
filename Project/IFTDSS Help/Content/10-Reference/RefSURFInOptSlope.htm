Surface Fire Behavior Input Options – Slope

Name Units Brief Description
Slope is specified Degrees or pPercentage You have two options:
  • as degrees.

    With this option, slope is entered as degrees (100 times the tangent of the slope in degrees). A 100% slope has a 45-degree incline.
    •
  • as percent.

    With this option, slope is specified as a percentage that is 100 times the slope rise divided by the slope reach. A 100% slope has a 45-degree incline.
Slope steepness If choosing map measurements, the units are ratio, feet, inches, digits (1-100); or ratio, meters, centimeters, digits (1-100)

If choosing worksheet, the units are in pPercentage or degrees 		You have two options:
  • is calculated from map measurements.

    With this option, slope steepness is calculated from a map representative fraction, contour interval, map distance, and number of contours crossed.
    •
  • is specified on the worksheet.

    With this option, you enter slope steepness directly.
