Fire Models

IFTDSS is a model integration framework containing a library of available software models.

This section contains pages for each fire model supported by IFTDSS. For each model, there are pages containing a brief overview, a link to a PDF containing information specific to each model, and pages providing information about the input and output options and parameters.

As needed, we have provided citations for works from which models have been derived, with links to the Bibliography, which contains the full text for each citation and, wherever possible, links to the referenced documents online.
