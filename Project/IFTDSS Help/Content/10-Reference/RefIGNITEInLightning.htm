Input Parameters – Probability of Ignition from Lightning

Name Units Brief Description
100-hr Fuel Moisture Percentage The portion of the 100-hour timelag dead fuel that is water, calculated on a dry weight basis. The 100-hour fuel moisture content has less influence on the fire spread calculations than the 1-hour or 10-hour moisture content do.
Lightning Duff and Litter Depth Inches, centimeters The average depth of the litter and duff layers at the point of the lightning strike. This value is used to calculate the probability of ignition from lightning. The word "lightning" is included in the parameter name to emphasize that this duff and litter depth is not used in any other model.
Lightning Ignition Fuel Type A drop-down list Lightning ignition fuel type is used to calculate probability of ignition from a lightning strike. There are eight fuel types; these types are only used for this ignition model.
Lightning Strike Type A drop-down list The type of lightning discharge (polarity). This value can be negative (-), positive (+), or unknown (?).
