Spotting Distance Output Parameters – Burning Piles

Name Units Brief Description
Cover Height Applied for Spotting Distance from a Burning Pile Feet, meters Cover height applied for spotting distance from a burning pile is the cover height actually used in the computation of spotting distance. A critical cover height is first determined from the maximum firebrand height. The applied cover height is the greater of the critical cover height and the input canopy height.
Firebrand Height from a Burning Pile Feet, meters Firebrand height from a burning pile is the maximum height to which a firebrand can be lofted and still be viable for starting a fire when it lands.
Flat Terrain Spotting Distance from a Burning Pile Miles, kilometers The flat terrain spotting distance from a burning pile is the maximum spotting distance over flat terrain. This value is then adjusted according to the terrain descriptors.
Spot Distance from Burning Pile Miles, kilometers The maximum distance at which one can expect potential spot fires resulting from firebrands from the burning pile. Flame height from a burning pile is an input used to calculate the lofting strength from the burning pile.
