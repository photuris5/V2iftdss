Containment Resources (IFT-contain)

You can use the IFTDSS fire containment module to estimate the fire suppression and containment resources necessary for containing a fire that is growing from a point source. The fire containment module implemented in IFTDSS produces the same results (which may differ slightly due to rounding) as the containment module in BehavePlus.

  • In order to run this module, you'll need information on resources and fire size to provide as inputs.
  • Output variables include resource information and information about the fire (e.g., area, perimeter, and containment status).
