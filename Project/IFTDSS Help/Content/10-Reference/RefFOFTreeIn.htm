Tree Mortality Input Parameters

Name Units Brief Description
Crown Ratio A ratio The ratio of live crown length to total tree height. In this system, whole number codes are used (for example, 4 indicates that the live crown length is 40% of tree height).
Diameter Breast Height Inches, centimeters Diameter at breast height expressed in inches or centimeters. Breast height is 4.5 ft or 1.37 meters above the ground.
Fire Severity drop-down choice Fire severity: 1 – low, 2 – moderate, 3 – high, 4 – very high, 5 – extreme.
Flame Length Feet, meters Flame length is the distance from the ground at the leading edge of the flame to the tip of the flame.
Scorch Height Feet, meters The average height of foliage browning or bole (tree trunk) blackening caused by a fire.
Tree Height Feet, meters The above-ground height of individual trees, or groups of trees, from the tree base to tree top. Tree height can be expressed in feet or meters.
Tree Species NA The species name of the individual tree(s) in question.
Trees per Acre Trees per acre The stand density (number of individual trees per unit area) expressed in trees per acre.
Use flame length or scorch height? drop-down choice This choice provides the user with the option of using either flame length or scorch height in the tree mortality equations.
