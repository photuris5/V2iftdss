Output Parameters

Details regarding output parameters are provided in subsequent tables for

  • Spotting distance from a burning pile. Intermediate values include cover height, firebrand height, and flat terrain spotting distance.
  • Spotting distance from torching trees. Intermediate values include cover height, firebrand height, flat terrain spotting distance, steady state flame duration, steady state flame height, tree height/flame height ratio.
  • Spotting distance from a wind-driven surface fire. Intermediate values include cover height, firebrand drift distance, firebrand height, and flat terrain spotting distance.
