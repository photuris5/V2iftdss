Surface Fire Behavior (FCCS) Inputs – Environmental Parameters: Custom Fuel Moisture Scenarios

On the Custom Fuel Moisture page, you can enter multiple custom moisture scenarios and save them with your fuelbed. You can select one custom moisture scenario for each run of the FCCS module. In contrast to the standard fuel moisture scenario (in which IFTDSS assigns the moisture values), with a custom fuel moisture scenario, you assign the values you wish to use for each parameter.

Name Units Brief Description
1-hour Percentage 1-hour fuel moisture (0.0 to 0.25 inch diameter woody fuels).
10-hour Percentage 10-hour fuel moisture (0.25 to 1.0 inch diameter woody fuels).
100-hour Percentage 100-hour fuel moisture (1.0 to 3.0 inch diameter woody fuels).
1000-hour Percentage 1000-hour fuel moisture (3.0 to 9.0 inch diameter woody fuels).
Crown foliar moisture Percentage Live foliar moisture of tree crowns.
Description NA Description of moisture scenario. You enter the description.
Duff fuel moisture Percentage Duff fuel moisture.
Name NA Name of moisture scenario. You create this name.
Nonwoody Percentage Live foliar moisture of nonwoody fuels.
Shrub Percentage Live foliar moisture of shrubs.
