Surface Fire Behavior (FCCS) Inputs – Woody Fuels: Stumps

There are three subcategories of stumps:

  1. Sound: Stumps composed of sound wood.
  2. Rotten: Stumps composed of rotten wood.
  3. Lightered: Stumps with resin-soaked heartwood.

 

Name Units Brief Description
Density Number per acre Number of stumps per unit area in each category.

Min: minimum density.
Mode: most common density.
Max: maximum density.
Diameter Inches Diameter of stumps in each category.

Min: minimum diameter.
Mode: most common diameter.
Max: maximum diameter.
Height Feet Height of stumps in each category.

Min: minimum height.
Mode: most common height.
Max: maximum height.
Relative cover Percentage Relative percentage cover of species in all stump categories. Must sum to 100%.

Min: minimum percentage cover.
Mode: most common percentage cover.
Max: maximum percentage cover.
Scientific name A drop-down list Scientific (Latin) name of species in each stump category. You can select a maximum of ten species.
