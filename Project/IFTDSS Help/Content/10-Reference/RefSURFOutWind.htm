Surface Fire Behavior Outputs – Wind

You can select one or more of these options.

Name Units Brief Description
Crown fill portion Percentage An intermediate value in calculating the WAF. Crown fill portion is calculated from canopy cover, canopy height, and crown ratio. If crown fill portion is less than 5%, the calculation for unsheltered WAF is used; otherwise, the calculation is for sheltered WAF. If WAF is a model input, the value for crown fill portion is not meaningful and is set to zero.
Crown ratio A ratio The proportion of the total vertical tree height that is occupied by the vertical length of the tree crown. Crown ratio and canopy height are used to find canopy base height, which is used in the IFT-surfacemodule to calculate WAF.
Effective wind speed limit Miles per hour, kilometers per hour A function of reaction intensity.
Effective wind speed Miles per hour, kilometers per hour The combined effect of midflame wind speed and slope steepness in the direction of maximum spread (head fire). This effective wind speed is used to determine the shape (length-to-width ratio) of a point source fire. Calculation of effective wind speed is based on the wind and slope factors that are part of Rothermel's fire spread model (1972).
Maximum effective wind exceeded Yes or No Indicates whether the effective wind speed limit, as calculated in the surface fire spread model, has been exceeded. You have the option of imposing the maximum reliable effective wind speed. If Yes, the maximum reliable effective wind speed from Rothermel (1972) is used in the calculations. If No, the input values are used. (Albini, 1976b)
Midflame wind speed Miles per hour, kilometers per hour The wind speed that exists at midflame height above the fuelbed and affects surface fire spread. Midflame wind is often called eye-level wind.
Wind adjustment factor calculation Indicates the method of calculating WAF. If the crown fill portion is less than 5%, the unsheltered model is used; otherwise, the sheltered model is used. If WAF is a model input rather than a calculated value, the variable is not meaningful and is set to Input.
Wind adjustment factor A value between 0 and 1 A value between 0 and 1 used to adjust the wind speed at 20-ft above the vegetation to midflame wind speed. Midflame wind speed = 20-ft wind speed x WAF. (Albini and Baughman, 1979; Baughman and Albini, 1980; Rothermel, 1983)
