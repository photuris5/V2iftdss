Surface Fire Behavior (FCCS) Inputs – Ground Fuels: Squirrel Midden

Squirrel middens are mounds of litter, tree cones, and other plant material accumulated by squirrels; this type of ground fuel is only applicable to a few forested fuelbeds in Alaska.

Name Units Brief Description
Density Number per acre Number of squirrel middens per acre.
