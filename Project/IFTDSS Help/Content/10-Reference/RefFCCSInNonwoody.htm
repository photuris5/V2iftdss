Surface Fire Behavior (FCCS) Inputs – Nonwoody Fuels

Nonwoody fuels are categorized into two layers:

  1. Primary nonwoody layer: Main vegetation layer distinguished by height, form, species composition, or any other user-defined characteristic.
  2. Secondary nonwoody layer: Secondary user-defined vegetation layer.

 

Name Units Brief Description
Height Feet Height of vegetation in each nonwoody vegetation layer.

Min: minimum height.
Mode: most common height.
Max: maximum height.
Live foliar moisture content Percentage Live foliar moisture content in each nonwoody vegetation layer.

Min: minimum live foliar moisture content.
Mode: most common live foliar moisture content.
Max: maximum live foliar moisture content.
Loading Tons per acre Fuel loading (biomass) of nonwoody vegetation for each layer.

Min: minimum loading.
Mode: most common loading.
Max: maximum loading.
Percent cover Percentage Percentage cover of nonwoody fuels by crown projection of each nonwoody vegetation layer.

Min: minimum percentage cover.
Mode: most common percentage cover.
Max: maximum percentage cover.
Percent live Percentage Percentage of biomass that is live in each shrub layer.

Min: minimum percentage live.
Mode: most common percentage live.
Max: maximum percentage live.
Relative Cover Percentage Relative percentage cover of species in each nonwoody vegetation layer. Must sum to 100%.

Min: minimum percentage cover.
Mode: most common percentage cover.
Max: maximum percentage cover.
Scientific name A drop-down list Scientific (Latin) name of each species in each nonwoody vegetation layer. You can select a maximum of 10 species for each nonwoody layer.
