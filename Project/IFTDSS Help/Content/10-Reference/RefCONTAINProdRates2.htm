Fireline Production Rates for Initial Action by Hand Crews

This table presents fireline production rates for initial action by hand crews in chains per person per hour. Table taken from National Wildfire Coordinating Group, 2004.

   Important   

These rates are to be used for estimating initial action productivity only. Do not use these rates to estimate sustained line construction, burnout, and holding productivity. Initial action consists of scratch line construction and hotspotting.

This information is used for the Resource Line Production Rate parameter. For other fireline production rate information used for this parameter, see Sustained Fireline Production Rates, Fireline Production Rates for Initial Action by Engine Crews, Fireline Explosives Crews Production Comparison, Bulldozer Fireline Construction Rates, and Tractor-Plow Fireline Production Rates.
