Surface Fire Behavior (FCCS) Inputs – Canopy Snags

There are four subcategories of canopy snags:

  1. Decay Class 1 with foliage: Recently dead conifers that still have most of their foliage in addition to bark, branches, and tree top.
  2. Decay Class 1 without foliage: Recently dead trees that still have bark, branches, and tree top intact.
  3. Decay Class 2: Snags that have shed fine branches but retain coarse branches and bark.
  4. Decay Class 3: Snags with extensive heartwood decay and no bark, branches, or tree top.

 

Name Units Brief Description
Density Number per acre

Number of snags per acre for each class where

Min: is the minimum density,
Mode: is the most common density, and
Max: is the maximum density.
Diameter Inches

Diameter at breast height (or highest point if the tree is broken below breast height) of snags for each class.

Min: minimum diameter.
Mode: most common diameter.
Max: maximum diameter.
Height Feet

Height of snags for each class.

Min: minimum height.
Mode: most common height.
Max: maximum height.
Relative cover Percentage Relative percentage cover of each species in each snag class. The sum must total 100%.

Min: minimum % cover.
Mode: most common % cover.
Max: maximum % cover.
Scientific name A drop-down list Scientific (Latin) name of species in each snag class. You can select a maximum of two species per snag class.
