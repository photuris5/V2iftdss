Spotting Distance (IFT-spot)

You can estimate the maximum distance from different possible sources where one can expect potential spot fires, then use that information when preparing a prescribed burn plan. The spotting distance models in IFTDSS produce the same results (which may differ slightly due to rounding) as the spotting distance module in BehavePlus.

Outputs include spotting distances from burning piles, torching trees, and wind-driven surface fires. For fire potential from each type of spotting distance, there is a set of input and output parameters.

More information about these models (PDF)

Spotting Distance from a Burning Pile

Spotting distance from a burning pile is the maximum distance at which one can expect potential spot fires resulting from firebrands from a burning pile.

Input variables include terrain, wind speed, and flame height properties. The model provides spotting distance from a burning pile.

(Albini, 1981)

Spotting Distance from Torching Trees

Spotting distance from torching trees is the maximum distance at which one can expect potential spot fires resulting from firebrands from torching trees.

Input variables include terrain, wind speed, flame height, and tree characteristics. The model provides spotting distance from torching trees.

(Albini and Baughman, 1979; Chase, 1981)

Spotting Distance from a Wind-Driven Surface Fire

Spotting distance from a wind-driven surface fire is the maximum distance at which one can expect potential spot fires resulting from firebrands from a spreading wind-driven surface fire.

Input variables include terrain, wind speed, and flame height. The model provides spotting distance from a wind-driven surface fire.

(Albini, 1983a; Albini, 1983b; Chase, 1984)
