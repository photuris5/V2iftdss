Reference Section

This reference section contains the following reference material:

  • the bibliography, which includes references for the citations in the IFTDSS help
  • a section on the 53 fire behavior fuel models supported by IFTDSS, including a summary table of the individual fire behavior fuel models, with a photo of a typical environment for each model, plus a matrix showing which moisture inputs for these fire behavior fuel models are required for each fire model in IFTDSS
  • a section on vegetation data sets, where you will find information on the supported vegetation data sets
  • an extensive section on fire models and modules. IFTDSS is a model integration framework containing a library of available software models and modules. For each model and module, there is a page of overview information, a PDF with additional informaiton (if applicable), information on the input and output options and parameters, and relevant citations for works from which models have been derived.
  • a section on other tools, which provides additional information about tools that may be useful, such as the Digital Photo Series and a unit conversion calculator

For each model, as needed, we have provided citations for works from which models have been derived, with links to the Bibliography, which contains the full text for each citation and, wherever possible, links to the referenced documents online.
