Surface Fire Behavior (FCCS) Inputs – Shrubs

  1. Primary shrub layer: Main shrub layer distinguished by height, form, species composition, or any other user-defined characteristic.
  2. Secondary shrub layer: Secondary user-defined vegetation layer.

 

Name Units Brief Description
Height Feet Height of vegetation in each shrub layer.

Min: minimum height.
Mode: most common height.
Max: maximum height.
Is needle drape sufficient to affect fire behavior? Yes or No Is the needle drape on the shrub component sufficient to affect the rate of spread (i.e., fire behavior)?
Live foliar moisture content Percentage Live foliar moisture content in each shrub layer.

Min: minimum live foliar moisture content.
Mode: most common live foliar moisture content.
Max: maximum live foliar moisture content.
Percent cover Percentage Percentage cover by crown projection in each shrub layer.

Min: minimum percentage cover.
Mode: most common percentage cover.
Max: maximum percentage cover.
Percent live Percentage Percentage of biomass that is live in each shrub layer.

Min: minimum percentage live.
Mode: most common percentage live.
Max: maximum percentage live.
Relative Cover Percentage Relative percentage cover of species in each shrub layer. Must sum to 100%.

Min: minimum percentage cover.
Mode: most common percentage cover.
Max: maximum percentage cover.
Scientific name A drop-down list Scientific (Latin) name of species in each shrub layer. You can select a maximum of 10 species for each shrub layer.
