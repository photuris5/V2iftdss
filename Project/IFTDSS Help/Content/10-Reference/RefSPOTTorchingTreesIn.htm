Spotting Distance Input Parameters – Torching Trees

Name Units Brief Description
20-ft Wind Speed (Upslope) Miles per hour, kilometers per hour The wind speed at 20 feet above the vegetation. In this module, the 20-foot wind speed is the wind that carries the embers.
D.B.H. Inches, centimeters The diameter at breast height of a tree measured at about 4.5 feet (1.4 meters) from the ground. For this parameter, enter an average DBH for the trees in the area.
Downwind Canopy Height Feet, meters Used to calculate spotting distance. It is the mean cover height in the downwind direction, as cover height affects the distance a firebrand is carried. If the forest cover is open, a value of half the downwind canopy height should be entered.
Number of Torching Trees Digits The number of trees that torch simultaneously. This value is used to calculate the height to which firebrands are lofted.
Ridge-to-Valley Elevation Difference Feet, meters The vertical elevation difference in feet or meters between ridge top and valley bottom in the direction of terrain flow. A stylized terrain shape is used to calculate spotting distance.
Ridge-to-Valley Horizontal Distance Miles, kilometers The horizontal distance in miles or kilometers between the ridge top and the valley bottom in the direction of the wind flow. A stylized terrain shape is used to calculate spotting distance. If winds are blowing directly up a valley or down a valley, the ridge-to-valley horizontal distance most likely is zero.
Spot Tree Species A drop-down list Used in estimating the maximum spotting distance from torching trees. Fourteen tree species are available.
Spotting Source Location A drop-down list The origin of the firebrands that can result in spotting. Possible locations are
  • Ridge top (RT)
  • Midslope, windward side (MW)
  • Valley bottom (VB)
  • Midslope, leeward side (ML)
Torching Tree Height Feet, meters The distance from the ground to the top of a tree.
