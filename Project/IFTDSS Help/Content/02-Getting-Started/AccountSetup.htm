Setting Up Your Account

IFTDSS is a web-based software integration framework that can be accessed using most standard Internet browsers (i.e., Microsoft Internet Explorer or Mozilla Firefox).

  • Accessing IFTDSS
  • The Home Page
  • Creating a User Profile

Accessing IFTDSS

To access IFTDSS, use an Internet browser to navigate to http://iftdss.sonomatech.com.  This takes you to the IFTDSS main page. 

To set up a user account, choose Request Account on the left portion of the main page.

To obtain the IFTDSS access code, please send an email to IFTDSSRequest@sonomatech.com.  The access code will be emailed to you.

Once you have the access code, enter a user name, password, and the access code to set up an account. Your user name can contain spaces. After setting up your account, you can log in.

The Home Page

After you set up an account and log in, you are taken to the Home page. A menu of options is displayed on the left side of the screen. Because this is your first time logging in, IFTDSS has a few suggestions: create a profile and take a look at the "getting started" materials.

To create a user profile, click the link on the right side of the page.

Creating a User Profile

After you create a user profile, others can identify you within the system. To start creating your profile, provide your first and last name and email address.  Your email address will only be used to send periodic IFTDSS-related updates, though other users will be able to see and use it to communicate with you. You can also upload a photograph or image to associate with your user profile. If you wish, fill in the other fields as well. Your name, agency, and location will appear in the list of IFTDSS users.

You can always edit your profile information later.
