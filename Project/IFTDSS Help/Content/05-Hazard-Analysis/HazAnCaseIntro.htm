Hazard Analysis Tutorials

To illustrate how to use the hazard analysis workflow in IFTDSS, we have prepared some tutorials that walk through some common activities.

The first tutorial explains how to use IFTDSS to perform a landscape-scale hazard analysis to identify areas across a landscape that may be potentially hazardous if a fire were to occur. This tutorial shows how to

  • Acquire LANDFIRE data and set up a project in IFTDSS
  • Create a run to model fire behavior across a landscape (IFT-FlamMap)
  • Examine data relative to points of interest and other geographic features using Google Earth
  • Identify areas of potential fire hazard across a landscape

The second tutorial, which you will find under Prescribed Burn Planning, explains how to use the hazard analysis tools in IFTDSS for fire contingency planning to support prescribed burn planning.

In addition, the following materials can be helpful:

  • a webinar, Using IFTDSS for Hazard Analysis, which was presented on March 28, 2012
  • a quick tutorial, Calculating fire behavior across a landscape (IFT-FlamMap)
