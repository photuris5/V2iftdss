Overview of the Modules in IFTDSS

IFTDSS is a model integration framework; it contains a library of software modules with a variety of practical applications in the fire sciences. The modules in IFTDSS provide information for calculating fire behavior, effects, containment, probability of ignition, risk, and other fire-management-related information.

Within the IFTDSS interface, modules are presented in three ways:

  • by IFTDSS-defined workflows;
  • by the model development team and type; and
  • by a listing of all available modules in the system.

The Reference section provides detailed information on fire module inputs and outputs.

Organization by Workflows

With the modules organized by workflow, you can

  • Prepare a prescribed burn plan. You can use IFTDSS to create a burn plan document, predict fire behavior, predict fire effects, and perform contingency planning. Some of the information from a hazard analysis can also be incorporated into the development of a burn plan document. See modules for prescribed burn planning for a list and brief description of the modules used in this pathway.
  • Perform a landscape-scale hazard analysis. The landscape-scale hazard analysis can help identify areas across a landscape that could be hazardous if a fire were to occur. The information provided from fire hazard analysis tools is helpful for prioritizing where treatments may be warranted and for use in prescribed burn planning. The hazard analysis tools can also be used for assessing the areas adjacent to a burn location. See modules for hazard analysis for a description of the modules used in this pathway.
  • Perform a risk assessment. Using IFTDSS, you can perform a risk assessment using two different approaches. The results from the risk assessment can provide information useful for evaluating and prioritizing where to place fuel treatments to reduce fire hazard and risk to valued resources. For more information, see the risk assessment section.

Organization by Model Team and Type

The models and modules organized by model team include all of the same modules and pathways used in the IFTDSS workflows.

The types fall under the category of Fire and Fuels Application (FFA) tools:

  • FCCS fuel characteristics
  • FCCS fire potentials
  • FCCS surface fire behavior
  • consumption and emissions

See FFA model and module types for more information on these tools.

   Note   

A module makes available the information calculated by one or more models.
