Downloading LANDFIRE Data to IFTDSS

This section shows how to download a LANDFIRE data set to IFTDSS and then use that data in a project and run.

  1. On the Saved Data Sets page, you may acquire LANDFIRE data from either the LANDFIRE/LCP data tab, or the Fuelbed Data tab. choose Acquire Data from LANDFIRE (a link in the upper left corner).
  2. This choice takes you to the Acquire Data from LANDFIRE page, where you name your data set and select an area of interest for your project using the map window. To set your area of interest, use one of these two options:
    1. Select Draw Box (a radio button along the top left of the map window) to manually draw your area of interest on the map. When you use this option, IFTDSS automatically enters the geographic coordinates in the coordinate entry boxes (the North, East, South, and West fields in the center top of the page).
    2. Alternatively, enter the geographic coordinate extents of your area of interest using the coordinate entry boxes.

  3. Once you have defined your area of interest, choose Next. LANDFIRE data for that area is retrieved.

       Note   

    The area of interest for your data set will automatically set the area of interest for a project when you choose to use the data set for a new project.

  4. After your data set has been successfully acquired, you are returned to the Saved Data Sets page, where a message confirms that your data was successfully acquired. Your new data set is in the list of available data sets. Because you created the data set directly and not as part of a project, there is no project associated with the data set yet.


    As with any other data set, you can edit, delete, rename, or copy the data set.

  5. To use this data set for a project, choose the Projects tab (at the top of the page) and select an existing project, or choose Home > Create a New Project.

  6. From the Projects tab, choose Create New Project (the link at the upper right of the page). The Create New Project page appears, where you can give your project a name and enter optional information. (For more information on creating a project, see Creating a Project.)
  7. After filling in your project information, choose Next to start the process of setting up a new run for this project. In this example, you will be using the LANDFIRE data you just acquired. For this run, we will perform a hazard analysis across a landscape to identify areas that could be hazardous if a fire were to occur.
  8. Make the following selections: By IFTDSS Workflows > Hazard Analysis > Fire Behavior > Calculate fire behavior across a landscape (IFT-FlamMap). This takes you to the following page.

  9. To use the LANDFIRE data set that you just imported, select the second option (Use an existing data set) and use the drop-down selector to select your data set. Choose Next to set up a new run. This takes you to the Create New Run page
  10. On the Create New Run page, you can specify a name for your run.
  11. Choose Next. This takes you to the first page in the pathway. Here you can view the steps for this particular pathway along the top of the page. The boxes along the top of the page can be used to navigate through a module run. At the bottom of the screen, you have the option to choose a different data set for the run. In this example, we will use the data set that was previously chosen.

  12. Choose Next. This takes you to the data input screen. On the data input screen, you may use the default data that has been pre-populated or you may change any of the input values. For more on the data screen, see Providing Input Data.
  13. Choose Next. This takes you to the review spatial input data screen. Here you can view the LANDFIRE data that you have chosen to use for this run.

       Note   

    If you would like to edit the LANDFIRE input data, access the Project Summary page (by clicking on the breadcrumb navigation at the top left of the page), scroll to Project Data, and choose Edit. This launches the spatial data editing tools. Once you have edited your data, return to the Project Summary page and choose to continue the run that you have just set up.

  14. When you are ready to run this module with this data, choose Next. IFTDSS runs the module with the data set, then displays the results. Here you can view the output parameters from the fire behavior module on a map.

  15. Choose Finish to return to the Project Summary page.
