Creating a Run

Once you've created a project, the next step is to create a new run. A run defines the analysis activity that you would like to perform within the project area, and also defines the models, tools, or analysis steps that you will use.

At a high level, the activities you will be performing to create a run within a project are to

  1. Choose the type of run you wish to make.
  2. Select the module you wish to run.
  3. Name your run. Run names must be unique.
  4. Configure the module.
  5. Provide the input data required by that module.
  6. View your output data.
  7. View your run summary.

IFTDSS saves your run so you can view it later.

The sections on this page walk through the steps of creating a run for prescribed burn planning. These steps follow the process of creating a project. For an overview of more detailed tutorials on prescribed burn planning, as well as on the other types of workflows in IFTDSS; see Getting Started.

Choosing the Type of Run You Wish to Make

To create a new run, start by choosing the type of run you wish to make. IFTDSS provides tools for prescribed burn planning, hazard analysis, and risk assessment.

  • Using the tools available for prescribed burn planning, you can model fire behavior and fire effects and perform other types of analyses (i.e., fire containment and probability of ignition) that support the development of burn plan documentation.
  • Using the tools available for performing a hazard analysis, you can model fire behavior across a landscape and identify areas that could be hazardous if a fire were to occur.
  • Using the tools available for performing a risk assessment, you can obtain information useful for evaluating and prioritizing where to place fuel treatments to reduce fire hazard and risk to valued resources.

In this case, you are going to be creating a run in support of a prescribed burn plan project, so on the Projects tab, choose the desired project), then select Create New Run. This takes you to the page where you can choose how to access the modules in IFTDSS.

Because you know you are doing a prescribed burn plan run, choose By IFTDSS Workflows, which displays your choices.

 

Next, choose Prescribed Burn Planning, which opens a page listing different modules. In IFTDSS's Prescribed Burn Planning toolkit, there are several different modules you can use for prescribed burn planning and to develop a burn plan document that you can print out and edit (in Microsoft Word format).

 

Selecting the Module for Your Run

After selecting Fire Behavior, you will see a list of fire behavior modules that you can run.

Note

The box above the list of modules serves as a navigation tool and can be used to move back to previous pages.

 

Choose calculate fire behavior for individual stands (IFT-FlamMap) to model fire behavior for a single stand. This takes you to the next page.

Naming Your Run

On the next page, name the run that you are creating.

   Note   

Run names must be unique.

Choose Next.

Configuring the Module

After you name your run, the configuration page appears. In the top portion of this page are boxes that indicate where you are in the model pathway. Below that is a line with the name of your run and of the module you are running. In the area below that line are fields for entering specific information for the run.

In the first field, indicate the number of simulations you wish to run. Also, if desired, choose to Enable variable winds (a check box), then, in the second field, enter the number of different wind speeds you wish to use (this number is not the wind speeds themselves; just how many wind speeds you wish to use). Choose Next.

Note

The navigation graph at the top of the page indicates where you are in the pathway. You can navigate through the pathway by clicking on the graph.

Providing Input Data

On the data input page, provide the input data that are required by the module that you have chosen. The input forms are pre-populated with default data values; you can edit these values, including the wind speeds, by clicking in the entry boxes. For more information about the module input (and output) parameters, please see the Reference section.

Viewing Output Data

Choose Next on the input page. IFTDSS runs the module you have chosen and displays the outputs. There are three options for viewing most module outputs:

  1. table view (the default),
  2. graph view, and
  3. map view (which is only available when working with spatial data).

Table View

Table view lists the output parameters along the left side of the page and the parameter values to the right. To view all outputs, use the scroll bar along the bottom of the page.

To switch to graph view, click Graph (under Views).

Graph View

In graph view, the output parameters are listed along the left side of the page and the parameter values are displayed to the right in graphical form. In this example, for each fuel model that had been selected, rate of spread (y-axis) is graphed as a function of 20-ft wind speed (x-axis).

To save the graph as an image file to your computer, click Save Image (at the bottom left of the page).

Map View

When you are working with spatial data (for example, LANDFIRE gridded data), the inputs and outputs are displayed in map view.

The map view page lists the map layers along the left side of the map and the map legend along the right. The map layers can be toggled on and off by clicking the box next to each layer.

You can save images of different map views to your computer by choosing Save Map Image. You can export map layers and view them in Google Earth by choosing Download KML File.

Viewing the Run Summary

When you have finished viewing and working with the outputs from a run, click Finish (at the bottom of the page). The run summary page is displayed.

On the run summary page, you can

  • view general information about a run,
  • copy and save data sets used in the run, and
  • save the data used in and generated by the run to your computer.

If you are working on a burn plan document, this is where you can save the input and output data from a run to your computer and then import the information into your burn plan (as a Microsoft Word document).

Note

The Prescribed Burn Plan section contains a more detailed discussion of how to get information out of IFTDSS and into your burn plan document.

Use the breadcrumb navigation along the top left of the page to return to the Project Summary page that is associated with your run.
