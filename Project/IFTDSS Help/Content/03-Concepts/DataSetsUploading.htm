Uploading a Data Set to IFTDSS

To create a data set, you can upload a LANDFIRE landscape file (an .lcp file and accompanying .prj file), fuelbed dataset, or a Uploading a Shapefile to IFTDSS or an FCCS Fuelbed .xml file from your local computer. Once uploaded, these files are available for use in various IFTDSS modules and tools.

To upload a landscape (.lcp and accompanying .prj file)

  1. From the Home page, choose Manage My Data Sets or access the Data tab. This takes you to the Saved Data Sets page.
  2. Choose the LANDFIRE/LCP Tab and select Upload New Data Set.
  3. This opens the Upload New Data Set page.
  4. Enter a name for your data set. This name will later be displayed on the Saved Data Sets page.
  5. Add a file by choosing Choose File and browsing on your local computer for the file (in .lcp or .xml format; each .lcp file also needs a corresponding .prj file).
  6. Once you have found the desired file, choose Add File.
  7. If you want to add more files, repeat steps 2 through 5 for each additional file.
  8. Once you have selected all the files you want to upload, choose Finish to upload the files.

After you have uploaded your file(s), your new data set is listed on the Saved Data Sets page.
