Creating a Values-at-Risk Map Within the Risk Assessment Pathways

   Note   

If you did not navigate to this tutorial through one of the risk assessment pathways in IFTDSS, please refer to one of the more detailed risk assessment tutorials for information on the context within which you are creating a values-at-risk map.

This is a quick tutorial on how to create a values-at-risk map. This tutorial may seem to start in the middle of something; that is because it is intended to only be used as part of the risk assessment pathways when you are on the Defining Values at Risk step. If you have not gotten to this online help page by clicking a link on that step, please see the introduction to the risk assessment tutorials.
