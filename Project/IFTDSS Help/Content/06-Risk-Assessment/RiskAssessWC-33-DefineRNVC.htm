Step 7a: Understand How Relative Net Value Change is Defined

For this tutorial, we are going to select a pixel using the Point Edit tool.

The pixel shown in the illustration here has been assigned a Response Function 3, and the area represented by this pixel was predicted to have burned with a “High” flame length (8 to 11 ft); therefore the pixel is assigned a Net Value Change multiplier of -20 (see the following table, the topics Values at Risk and Response Functions and Values at Risk, and the Response Functions Table).

The overall burn probability for the grid cell is 0.199, so the burn probability of 0.199 is multiplied by the net value change multiplier of -20 to obtain a net value change value of -3.98.

The formula is

Net Value Change = BP x NVC Multiplier

Plugging in the values, we get

Net Value Change = (0.199) x (-20)

Net Value Change = -3.98
