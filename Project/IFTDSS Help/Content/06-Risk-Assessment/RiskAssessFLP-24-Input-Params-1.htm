Step 4: Input Environmental and Simulation Parameters

Now that you have created a values-at-risk map, the next step is to input IFT-FlamMap parameters (fuel moisture and weather), input IFT-FlamMap-RANDIG simulation information, and set your flame length classes. When you first arrive at the Inputs step in IFTDSS, the input fields are pre-populated with default values.

Each of these inputs is covered in the next few topic pages.

   Tip  

You can use the Tab key on your keyboard to navigate to the next input field.
