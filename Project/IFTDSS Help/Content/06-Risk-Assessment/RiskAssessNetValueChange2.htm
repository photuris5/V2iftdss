Example Using the Worst-Case Flame Length Approach

This topic provides an example of how net value change is calculated for a risk assessment using the worst-case flame length approach.

The pixel in the following image has been assigned a response function of 3.

The area represented by the pixel burned with a high flame length (8 to 11 feet; see Flame Length Interpretations); therefore, in accordance with the response function table, the pixel is assigned a net value change (NVC) multiplier of -20.

The overall burn probability for the grid cell in this example is 0.199. The burn probability of 0.199 is multiplied by the NVC multiplier of -20 to obtain a net value change value of -3.98.

Net Value Change = Overall Burn Probability x NVC Multiplier

Net Value Change = 0.199 x (-20)

Net Value Change = -3.98
