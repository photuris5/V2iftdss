Relative Net Value Change

Relative net value change is the main output of the risk pathways. The relative net value change in land value due to fire is assumed to be a function of fire behavior, and includes both positive and negative effects for fire.

The following topics provide more detailed explanations and examples of net value change, focusing on what it is, how it is calculated, and how it is used for risk assessment:

  • Example Using the Flame Length Probabilities Approach
