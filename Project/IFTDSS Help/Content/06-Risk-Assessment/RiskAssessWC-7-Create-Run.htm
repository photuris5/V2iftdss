Step 2a: Create a New Run

To create a new worst-case flame length run,

  1. Under Filter, choose Create New Run (indicated by the letter C in the illustration). In the next few steps, you will be choosing the type of run you want to create using the acquired data.
  2. Choose By IFTDSS Workflows.

  3. Choose Risk Assessment.

  4. Choose Risk Assessment – Worst Case Flame Length. The next step is to select an area of interest within the project boundary.
