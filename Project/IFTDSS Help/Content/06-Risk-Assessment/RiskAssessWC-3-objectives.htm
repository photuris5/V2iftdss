Tutorial Objectives

In this risk assessment tutorial, you will

  • Set up a project in IFTDSS and acquire LANDFIRE data for that project.
  • Create a run focusing on risk assessment using the worst-case flame lengths approach. The steps on this process are to
    • Develop a values-at-risk map, defining values at risk and assigning response functions.
    • Input environmental and simulation parameters.
      • IFT-FlamMap inputs
      • IFT-FlamMap-RANDIG inputs
    • Review/edit spatial landscape input data.
    • Analyze potential fire behavior output data.
    • Identify potential fire risks across a landscape.
      • Use relative net value change data
      • Export data to Google Earth
  • Save data sets and copy runs for re-use
  • Assess further needs

At the end of this tutorial, we present and discuss risk assessment caveats.
