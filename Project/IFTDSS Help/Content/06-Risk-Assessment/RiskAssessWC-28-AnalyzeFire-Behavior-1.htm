Step 6: Analyze Fire Behavior Output Data

Both the IFT-FlamMap and IFT-FlamMap-RANDIG modules are now running. The more random ignitions you place across a landscape, the longer this step takes to complete.

After the IFT-FlamMap and IFT-FlamMap-RANDIG modules finish running, you can review your fire behavior output variables using the map and the Point Info tool.

The fire behavior output variables include

  • Overall burn probability
    • defined in Step 6a: Use the Point Info Tool
    • analysis is described in Step 6b: Analyze Overall Burn Probability Output
  • Flame length
    • defined in Step 6a: Use the Point Info Tool
    • analysis is described in Step 6c: Analyze Flame Length Output

Using the map, you can also review the values at risk, response function, and fire behavior fuel model layers.

   Note   

Due to random ignitions, your burn probability output will look different from this example’s output.
