Step 6: Analyze Fire Behavior Output Data

The IFT-FlamMap-RANDIG module is now running. The more random ignitions you place across a landscape, the longer this step will take to complete.

After the IFT-FlamMap-RANDIG module finishes running, you can review your fire behavior output variables using the map.

The fire behavior output variables include

  • Overall burn probability (defined in Step 6a: Use the Point Info Tool)
  • Burn probability by each flame length class (also defined in Step 6a: Use the Point Info Tool)

You can also review the values at risk, response function, and fire behavior fuel model layers.
