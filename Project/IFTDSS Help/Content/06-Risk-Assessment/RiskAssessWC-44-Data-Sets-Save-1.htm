Step 10: Save Your Data Set

After you have analyzed your high-risk areas in Google Earth, return to your risk assessment run in IFTDSS.

  1. From the Map Summary page, choose Finish.

  2. You are taken to the Run Summary page. Here, you can save your input and output data sets for future use in other runs as described below. Saving your data sets is useful; for example, if you edited the spatial landscape data, you can reuse the data you edited.

To save a data set,

  1. Select Save As on the risk input data set.
  2. Enter a unique name for the data set copy.
  3. Choose OK.
  4. After you save your data set, you are taken to theData tab. Here, all your data sets are saved. You can use these data sets in other runs. For example, if you edited the spatial landscape data during your run, the edits are saved in this data set. Or if you create a new spatial run (i.e. Calculate fire behavior across a landscape (IFT-FlamMap)), you can select this data set on the Configure step of the run. In this case, you can use the IFT-FlamMap-RANDIG input data set that you've created in this tutorial in any spatial run.
