Assessing Fire Hazard for Prescribed Burn Planning

This PDF tutorialPDF tutorial (available in the online version of this documentationshows how to use IFTDSS to create a prescribed fire contingency plan that is based on potential fire hazard. This type of analysis helps you assess potential fire hazards within and adjacent to a prescribed burn unit and to develop a contingency plan using the tools in IFTDSS.

High fire hazard is expressed by high potential fire behavior (e.g., flame length, rate of spread, and fireline intensity) and/or undesirable fire effects (e.g., tree mortality and emissions).

The product of this analysis is a digital map of spatially explicit information about the variability of fire hazard within and adjacent to a burn unit. The results (maps and information) from this analysis can be included in a burn plan document.

For an example of a hypothetical burn plan that could have arisen from performing the actions demonstrated in this tutorial, see Tecuya - A Hypothetical Burn Plan Example Tecuya - A Hypothetical Burn Plan Example in the online help.
