Preparing a Prescribed Burn Plan

This PDF tutorial The PDF tutorial in the online help illustrates how to use IFTDSS to prepare a burn plan. Tools for preparing and generating a burn plan document in IFTDSS are based on the 21 required elements of a prescribed burn plan identified in the Interagency Prescribed Fire Planning and Implementation Procedures Reference Guide (U.S. Department of Agriculture and U.S. Department of the Interior, 2008).

The tutorial describes what is minimally acceptable for prescribed fire planning and implementation and uses the burn plan example in the RX-341 Prescribed Fire Plan Preparation course (Red Bull Prescribed Burn Plan).

The result of this tutorial is a burn plan document in Microsoft Word format that you can download to your local machine. For an example of what this burn plan looks like in Microsoft Word, see the Red Bull RX-341 Burn Plan topic, which you can view or download the Red Bull RX-341 Burn Plan topic, which you can view or download from the online help.

Important

Microsoft Word 2007 or later, or Word 2003 with the Microsoft Office Compatibility Pack, is required to open the burn plan document. You can download the compatibility pack at Microsoft.com.

Note that once you create a burn plan document in IFTDSS, it is stored within the system and can be re-used for subsequent documents.

There are also mapping tools within IFTDSS that can be used to develop vicinity and project maps and can be used for hazard analysis and contingency planning. See Assessing Fire Hazard for Prescribed Burn Planning for information on using hazard analysis tools in support of your prescribed burn planning.
