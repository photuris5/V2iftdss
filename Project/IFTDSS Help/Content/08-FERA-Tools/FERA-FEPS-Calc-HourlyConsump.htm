Hourly Consumption Rate

Hourly consumption rate is calculated as described in the following sections:

  • Flaming Phase Consumption
  • Smoldering Phase Consumption
  • Smoldering Adjustment
  • Residual Smoldering Adjustment
  • Total Consumption Rate
