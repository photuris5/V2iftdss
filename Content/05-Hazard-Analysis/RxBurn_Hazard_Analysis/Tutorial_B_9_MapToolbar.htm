Introducing the MapToolbar

The map toolbar, located at the top of the map, provides tools for viewing and editing data. In IFTDSS, hover your cursor over each tool for a brief description of that tool.

   Tip:   

Click on Fuel Models under Help dropdown menu to read a description of the environment and fuels for each module.
