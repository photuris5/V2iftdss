Tutorial B: Hazard Analysis for Prescribed Fire Planning

This prescribed fire planning tutorial covers:

  • Acquiring LANDFIRE data and setting up a project in IFTDSS.
  • Reviewing and editing spatial landscape data.
  • Creating a run focused on fire behavior across a landscape (IFT-FlamMap).
  • Establishing environmental parameters.
  • Reviewing spatial landscape input data and fire behavior output data.
  • Examining data relative to points of interest and other geographic features using Google Earth.
  • Establishing a potential burn unit based on modeled fire behavior characteristics.
  • Identifying potential fire hazard outside of a burn unit to help develop holding and contingency plans.
