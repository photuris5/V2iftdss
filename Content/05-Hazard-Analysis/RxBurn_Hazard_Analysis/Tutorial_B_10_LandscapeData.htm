Reviewing Spatial Landscape Data

You can view the project area by the following LANDFIRE data:

  • Fuel Model
  • Elevation
  • Slope
  • Aspect
  • Canopy Coverage
  • Canopy Height
  • Canopy Base Height
  • Canopy Bulk Density

You can also view the project area using imagery, topography, or street maps within IFTDSS. Toggle between these layers under the Base Layer section on the Layers tab.

In this example, you can see the project area classified by Fire Behavior Fuel Model.

 

   Note   

The area is dominated by the following fuel models:

Olive pixels = GS2(122): Moderate load dry climate grass-shrub

Grey pixels = NB1: Urban/Developed

Light blue pixels = TL3(183): Moderate load conifer litter
