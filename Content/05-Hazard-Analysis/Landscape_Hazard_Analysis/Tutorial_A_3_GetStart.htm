Getting Started

To begin, choose create a new project under the Home tab. Choose a descriptive project name. If desired, fill in the optional information.

Choose Next.

After creating a new project, you will see the page for creating a new run. The next step is to acquire LANDFIRE data, so we are going to navigate away from this page.

Access the project you created. In this example, we chose the Landscape Hazard Analysis link.

Now, we will acquire data from LANDFIRE.

Next, select Acquire data from LANDFIRE, then choose Next.
