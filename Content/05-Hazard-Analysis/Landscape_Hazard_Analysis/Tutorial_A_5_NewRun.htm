Creating a Run

After acquiring the LANDFIRE data, you are returned to the Project Summary page.

For use in spatial modules, you how have:

  1. Project Area of Interest
  2. Project Data Set

For your next step, choose Create New Run.

Choose the type of run you would like to create by choosing the following links:

  1. Hazard analysis
  2. Fire behavior
  3. Calculate fire behavior across a landscape (IFT-FlamMap)

In this run, you will

  1. Select an area of interest
  2. Input your environmental parameters
  3. Review the spatial landscape data
  4. Review the fire behavior output data

   Tip:   

Give your runs descriptive names for future reference.
