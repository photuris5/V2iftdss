Getting Started

The first step is to log into IFTDSS and create a new project by selecting the Create a New Project icon. If you already have existing projects and an established landscape you may create a new run from within your project to evaluate with MTT.

After selecting Create New Project a window similar to the one below will appear. Here you will name your project and fill in any optional project information that is appropriate and select the Next button. (You may choose your own information to populate the boxes.)

If you choose to work with an existing project, select the project from the Projects tab you wish to use and select Create New Run.
