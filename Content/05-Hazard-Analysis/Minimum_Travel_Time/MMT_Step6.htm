Configure

The next window is the Configure window, this is where the user may import data sets such as ignition, barrier or polygon files from other runs within the project. At this point in the tutorial we have only defined the landscape and we don’t have any data files to import. We will create these files in later steps. For now, ensure that the Squaw Creek data set is selected, and click Next.

   Note   

Once you select Next there will be a short duration data processing window before you are taken to the Review Landscape Data step.
