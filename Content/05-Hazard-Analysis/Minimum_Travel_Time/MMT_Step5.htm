Set Up Project Area of Interest

The next window is the Set up Project Area of Interest window. Here, the user will have several options.

  1. Chose a name for the data set using the Data Set Namebox.
  2. Choose a LANDFIRE version. LANDFIRE 2010 is the latest version available in IFTDSS and the default setting. This tutorial used LANDFIRE 2010
  3. Choose between the Scott and Burgan 40 and the Anderson 13 fuel models. This tutorial used the Scott and Burgan 40 fuel models
  4. Define a landscape extent by either:
    1. Selecting the latitude for the north and south extent and the longitude for the east and west extent and populating the appropriate coordinate boxes.
    2. Or drawing a box to define the landscape extent directly on the map by choosing the Draw Box radio button.

If using the Draw Box method the user would place the cursor at the top left-hand corner of the desired area and clicking the left mouse button and holding the mouse button drag the box to the lower right-hand corner of the desired area and release the mouse button. If a mistake is made in drawing a box simply reposition the cursor and repeat the process until you are satisfied with the extent.

Users can manipulate the map view by selecting the Navigate Map radio button at the upper left of the map. This makes the navigation tools active.

  1. Pan by clicking the left mouse button and holding it, and sliding the mouse in the direction you wish to move the map.
  2. The zoom in and out, + and – buttons can be used to change the map view extent.
  3. The arrows can also be used to pan around the map view.

If you wish to return to the North American extent simply delete the north, south, east and west coordinates in the boxes and the North American view will appear again.

Clicking the world button will take you to a full world view.

The project extent is not final until the Next button is selected. Keep in mind IFTDSS is limited to about a 400,000 acre landscape extent. Your landscape size is displayed at the upper right of the map view. Larger landscape extents require more computational processing thus more time for processing each run.

If you are adventurous chose your own landscape, otherwise you may chose the same landscape used in the tutorial by selecting North: 44.9577528, West: -109.7485415, East: -109.5630013, South: 44.8257449. Be sure to use the – (negative) at the beginning of your west and east coordinates.
