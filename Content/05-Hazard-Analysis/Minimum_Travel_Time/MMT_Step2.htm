Creating a New Run

Following the creation of a new project, or selecting the option to create a new run, you will be prompted to Choose the type of run you would like to create. Users are required to choose a pathway to access the type of run they would like to evaluate. You may access MTT through either the ”Hazard Analysis or Fuels Treatment pathways. The hazard analysis pathway allows a single MTT run while the Fuels Treatment pathway allows the user to run both a pre and post-treatment analysis within the same run that can be compared on a single map page. For this tutorial, we will choose the Fuels Treatment pathway.

Choosing either pathway will require clicking through several steps. For this example, choose the Fuels Treatment pathway>Fuels Treatment across a Landscape> LANDFIRE to be taken to the list of available runs.
