Review Landscape Data and Create a Fuel Treatment Polygon

After configuring the run we will go to the Review Landscape Data window. At this point we can assess and modify the LANDFIRE data to suit local conditions. The Landscape Statistics tool can be a powerful tool for assessing the landscape features. The Point Edit tool can be used to edit one pixel at a time. The user may also draw polygons to be used as masks to modify LANDFIRE data such as a recent fire or completed fuels treatment. You may also draw polygons to be used later as fuel treatment units. You may also choose to draw your treatment polygons later on the Pre Treatment Output window. You may also add notations or labels to the map using the Map Annotation tool .

Use the Draw Polygon tool to create the Squaw Creek Rx polygon on the landscape as shown on the map below. Use the Map Annotation tool to label it.

 

This polygon will be available to use for the remainder of your run. If you would like to save the polygon in your IFTDSS Datasets to use in other runs, scroll down the Save Polygon As dialogue at the bottom, and give it a descriptive name. It will saved for use in other runs upon clicking Next.
