Select Dataset

When IFT-MTT is chosen, you will be prompted to Select a dataset and an area of interest for your project. From this window you may acquire new data from LANDFIRE, specify an existing dataset to use, or upload a dataset from outside IFTDSS, such as a locally derived data set. In this tutorial we will use IFTDSS to acquire landscape data from LANDFIRE (If you chose to use an existing data set select that radio button and select the data set from the dropdown box). To proceed, click Next.
