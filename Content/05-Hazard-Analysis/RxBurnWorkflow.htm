RxBurn Planning Workflow

The tools available for prescribed burn planning can be used to model the probability of ignition, fire behavior and fire effects, and fire containment. These tools can also be used to develop a prescribed burn plan as outlined in the Interagency Prescribed Fire Planning and Implementation Procedures Guidelines and in the RX-341 Prescribed Fire Plan Preparation course.

The RxBurn Planning workflow is extensive. Click on the capability to learn more about the tools and pathways associated with it:

Probability of Ignition

Probability of Ignition

These tools can be used to calculate the probability of ignition from a firebrand or from a lightning strike.

Probability of ignition is calculated using the IFT-Ignite tool. IFT-Ignite, based on BehavePlus calculates the the likelihood that either a firebrand or lightning will ignite ground fuel of the type specified.

Firebrand Inputs for IFT-Ignite include:

  • 1-hr fuel moisture
  • air temperature
  • fuel shading from the sun

Lightning Inputs for IFT-Ignite include:

  • 100-hr fuel moisture
  • Duff and litter depth
  • Ignition fuel type
  • Lightning strike type
   Note   

If you are running this module for the IFTDSS burn plan document, these models can be used to fill in Elements 7, 16, and 17 of the burn plan.

 Fire Behavior

Fire Behavior

The tools available for modeling crown and surface fire behavior can be run for an individual stand or spatially using LANDFIRE (.lcp) data. The surface fire behavior module for FCCS fuelbeds requires FCCS fuelbed data as an input.

Options for calculating fire behavior include:

  • Calculate crown fire behavior (using IFT-Crown)
  • Calculate fire behavior across a landscape (using IFT-FlamMap)
  • Calculate fire behavior for individual stands (using IFT-FlamMap)
  • Calculate minimum travel time (using MTT)
  • Calculate surface fire behavior (using FCCS)
  • Calculate surface fire behavior (using IFT-Surface)
  • Predict surface fire behavior, size, and spread distance (using IFT-Surface+Size)
  • Calculate surface fire behavior for multiple fuelbeds using a single scenario and FCCS.

To calculate surface fire behavior using FCCS, the Fuelbed Crosswalk option is available, which derived FCCS fuelbeds from LANDFIRE Existing Vegetation Type.

   Note   

If you are running this module for the IFTDSS burn plan document, these models can be used to fill in Elements 7, 16, and 17 of the burn plan.

 Fire Containment

Fire Containment

The fire containment tools can be used to complete Elements 7, 16, and 17 of the burn plan.

With the Containment pathway, users can: 

  • Calculate safety zone size (using IFT-Safety)
  • Calculate spotting distance from a burning pile (using IFT-Spot)
  • Calculate spotting distance from a wind-driven surface fire (using IFT-Spot)
  • Estimate containment resources (using IFT-Contain)
  • Predict fire size and spread distance (using IFT-Size)
  • Predict surface fire behavior, size, and spread distance (Using IFT-Surface+Size)

 

   Note   

If you are running this module for the IFTDSS burn plan document, these models can be used to fill in Elements 7, 16, and 17 of the burn plan.

 Fire Effects

Fire Effects

The fire effects tools can be used to calculate consumption, emissions, tree mortality, and crown scorch height.

Using the Fire Effects pathway users can:

  • Calculate consumption and emissions (using IFT-FOFEM)
  • Calculate consumption and emissions across and landscape (using IFT-Consume)
  • Calculate tree mortality (using IFT-FOFEM)
  • Calculate fire effects from activity fuels or natural fuels using Consume and inputs from downloaded data
  • Calculate fire effects from activity fuels or natural fuels using Consume and manually inputting data
  • Predicting crown scorch height (using IFT-Scorch)
 Historical Fire Weather

Historical Fire Weather

The fire weather tools can be used to calculate historical daily and monthly fire weather statistics and fire danger indices based on the National Fire Danger Rating System (NFDRS).

 Create a Burn Plan

Create a Burn Plan Document

- The burn plan document pathway was designed based on the National Wildfire Coordinating Group's Interagency Prescribed Fire Planning and Implementation Procedures Guide (IPFPIPG, 2014) and consists of 21 elements (objectives, ignition plan, etc.).

Each screen in this pathway guides the user in developing the content for one of the 21 elements identified in the Guide. At any point during the development of the burn plan, the Prescribed Fire Plan can be downloaded as a Microsoft Word document using the Generate Burn Plan button at the bottom of any page in this pathway. The Prescribed Fire Plan will include all data that the user has entered and saved. Elements of the burn plan document include:

  1. Signature Page
  2. Part A: Agency Administrator Ignition Authorization
  3. Part B: Prescribed Fire Go/No-Go Checklist
  4. Complexity Analysis Summary
  5. Description of Prescribed Fire Area
  6. Objectives
  7. Funding
  8. Prescription
  9. Scheduling
  10. Pre-Burn Considerations and Weather
  11. Briefing Checklist
  12. Organization and Equipment
  13. Communication
  14. Public and Personnel Safety, Medical
  15. Test Fire
  16. Ignition Plan
  17. Holding Plan
  18. Contingency Plan
  19. Wildfire Conversion
  20. Smoke Management and Air Quality
  21. Monitoring
  22. Post-Burn Activities
  23. Appendices

After the user has completed the burn plan pathway and has downloaded the Prescribed Fire Plan, the user can easily update and/or customize all sections of the plan using Microsoft Word. Click here for more information about this module.
