Review

In this tutorial we walked through the steps needed to predict consumption and emissions using Consume. The tutorial provided information and step by step instructions on:

  • Setting up a project
  • Acquiring FCCS fuelbed data to run IFT-Consume
  • Selecting -IFT Consume from the IFTDSS workflows
  • Populating and running the IFT-Consume model
  • Viewing output data
  • Exporting data as raster layers to ArcMap for analysis
  • Evaluating results (in ArcMap)

Additional Help

To navigate to additional tutorials in the IFTDSS online help content,

  1. Click the Help button.
  2. Then select Getting Started (Tutorials and Videos) from the side menu.

On that page, you’ll find links to tutorials and videos on such topics as hazard analysis, prescribed burn planning, fuels treatment, spatial analysis across a landscape, and many more.
