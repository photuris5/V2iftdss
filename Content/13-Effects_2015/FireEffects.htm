Fire Effects Tutorials

The Fire Effects workflow offers runs to model fuel consumption, emissions production, tree mortality, and crown scorch. Consumption and emissions values can be obtained, in batch format, using IFT-FOFEM, or Consume for activity and natural fuels. Consumption and emissions results can be obtained geospatially for an area using the IFT-Consume. Tree mortality may be obtained using IFT-FOFEM, and crown scorch can be obtained using IFT-Scorch, a run derived from the Scorch module of BehavePlus.

 

As of 2015, these tutorials will address the consumption and emissions runs using FOFEM and Consume. Additional tutorials addressing tree mortality and crown scorch will be added later. Current tutorials include:
