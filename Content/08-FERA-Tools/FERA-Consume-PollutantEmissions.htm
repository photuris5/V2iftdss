Pollutant Emissions

Consume calculated total emissions for the following pollutant species:

  • PM (particulate matter)
  • PM10 (particulate matter < 10µm in diameter)
  • PM2.5 (< 2.5 µm)
  • CO (carbon monoxide)
  • CO2 (carbon dioxide)
  • CH4 (methane)
  • NMHC (non-methane hydrocarbons)

The rate of smoke production from fire varies by combustion phase. Less smoke is produced per ton of fuel consumed during the efficient flaming stage than during the less efficient smoldering phase (Peterson and Sandberg, 1988). Consequently, fuel consumption must be estimated by flaming and smoldering combustion phases for the best estimates of total emissions.  A 65% error is associated with emission calculations using total fuel consumption only rather than breaking the consumption into combustion phases (Peterson, 1987).

Consume calculates emissions based on a set of emission factors based on the type of fuel and the combustion phase of the burn. This table lists emission factors by fuel type (activity or natural), vegetation (Douglas fir, mixed conifer, ponderosa pine, hardwood, juniper, sage brush), and combustion stage (flaming, smoldering, and residual smoldering) (Hardy et al., 2001; Ottmar and Baker, 2007). The emission factors reported in Consume are calculated using the flaming and smoldering emission factors from that table and weighted according to the amount of fuel consumed in each phase.

Summary of Emissions Algorithm

Step 1:  Using input cover type to select an emissions factor group

Consume selects an emission factor group based on the input cover type. Many FCCS fuelbeds have more than one input cover type. FCCS outputs the first listed cover type to Consume for use in EFG assignments.

Step 2: Calculating emissions by combustion phase

Consume calculates pollutant emissions by combustion phase using the emission factors  by emisison factor group. In rare cases where an appropriate cover type is not available or is not listed for an FCCS fuelbed, Consume will use a default emissions factor (EFG #0).
