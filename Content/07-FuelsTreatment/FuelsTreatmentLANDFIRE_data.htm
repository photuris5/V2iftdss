Tutorial: Acquiring and Editing Spatial LANDFIRE Data

This tutorial describes how to add LANDFIRE data to IFTDSS and how to edit landscape files. In this tutorial, you'll learn how to

  • Upload a landscape (.lcp) file and corresponding projection (.prj) file from your local machine.
  • Acquire landscape data from LANDFIRE for a specified area of interest using map tools in IFTDSS.
  • Edit LANDFIRE data in Data Studio

You can view, print, or download this tutorial in PDF format a PDF version of this tutorial in the online documentation. After you've acquired your LANDFIRE data, read the tutorial on spatially evaluating fuels treatment across a landscape.
