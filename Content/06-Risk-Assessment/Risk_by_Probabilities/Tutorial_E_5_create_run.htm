Creating a New Run

To create a new run from the Project Summary page select Create New Run.

Flame Alternatively, you can navigate to your project from the Projects tab along the top Navigation bar.

Select the Risk Assessment workflow

Choose Risk Assessment – by Flame Length Probabilities

Define the Run Area

Name your run. 

   Tip   

For future reference, give your run a descriptive name.

In this step, select an area of interest within the project boundary. The run area may be the same size as the project area, or smaller. Smaller runs may be helpful of your interested in just one stand, treatment area, or unit within the project. For this example, we selected a smaller area within the project boundary by entering the following coordinates:

  • North: 38.155672364916
  • East: -122.5502871142
  • South: 38.120028050516
  • West: -122.5986956225

After selecting an area of interest, choose Next.

Selecting a Dataset

You are now on the Configure step.

Under Select Landscape Data, select the “Olompali State Historic Park (100%)” data set and choose Next to proceed to the Define Values at Risk step.
