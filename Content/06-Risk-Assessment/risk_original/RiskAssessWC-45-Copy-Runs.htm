Step 11: Copy Runs for Future Use

You can copy runs for use later. Copying runs is useful if you want to re-use a values-at-risk map. With copied runs, you can rerun the module using different inputs. This way, you can perform risk assessments across a variety of environmental conditions.

To copy a run,

  1. From the Data tab, navigate to the Projects tab.
  2. This takes you to the Active Projects page.
  3. On the Active Projects page, select the Risk Assessment - Olompali SHP project that you created at the beginning of this tutorial. This takes you to the Project Summary page.
  4. Find the run you want to copy, then choose the Copy link in that row. (For this tutorial, choose to copy the Olompali Risk run you've created.)
  5. Enter a name for the copied run and choose OK.
  6. After choosing OK, you are taken to the pathway of the copied run, where you can rerun the module using different inputs.

You have just a few more things to look at, and then you are done with this tutorial.
