Loading Rasters into ArcMap

Users may wish to view, analyze, or display outputs produced in IFTDSS in ArcMap.

You can download your data from your Project by clicking on the cog wheel under Actions and selecting Download.

The next section to the right from Actions is Export Status. You will now see a dropdown box for you to select if you'd like a raster or shapefile dataset. Select raster.

Once you have downloaded the raster file and extracted the data from the zip file to your computer, you can then load it into ArcGIS. One option for doing so is as follows.

Click on add data

Navigate to where you have the raster which will be downloaded as an image file.

Double click on the image file which will show you all the raster files available

Select the files you want to see and click add

You will see your raster data as shown
