Creating a Project

Because you use projects to organize your data, one of the first things you might want to do is create a project. A project is like a workspace. Each project has a geographic area of interest and may have one or more runs associated with it.

Follow the detailed steps below to learn how to create a Project, or watch this tutorial:

 

  1. Choose Next. Your project is saved and you are taken to a page where you choose the type of run you want to create. For this tutorial we will go to the project summary and acquire LANDFIRE data before selecting any runs. Click on the 'Squaw Creek' project link at the top left of the page:
