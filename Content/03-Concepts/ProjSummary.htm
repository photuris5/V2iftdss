Viewing a Project Summary

After you create a project, the project is saved and the Project Summary page appears.

All information about a project is stored on the Project Summary page, including

  • the project information (description, start and end date, and so on)
  • the project area of interest (or geographic extent)
  • the runs associated with the project
  • the project data sets used

From this page, you can

  • filter the view of runs and data sets
  • copy or delete runs
  • create new runs
  • edit, delete, rename, or copy project data sets
