Downloading LANDFIRE Data to IFTDSS

This section shows how to download a LANDFIRE data set to IFTDSS and then use that data in a project and run. It begins with how to reach the Acquire Data from LANDFIRE page via an existing Project, and via the IFTDSS homepage if you do not yet have an existing project.

Proceed to the instructions below for details on how to download LANDFIRE data into IFTDSS or watch this short instructional video:

 

 

Selecting LANDFIRE Data for an existing Project

From the IFTDSS home page, (A) click on the Projects tab, then (B) select the project for which the LANDFIRE data is needed from the Active Projects list. This will take you to the Project Summary page.

From the Project Summary page, select Acquiring data from LANDFIRE from the Area of Interest box on the right of the screen.

On the data selection page, select Acquire data from LANDFIRE, then click Next. This will take you to the Acquire Data from LANDFIRE page.

Selecting LANDFIRE Data for use in a Project that does not yet exist

From the IFTDSS homepage, select the Data tab to go to the Saved Data Sets page.

On the Saved Data Sets page, you may acquire LANDFIRE data from either the LANDFIRE/LCP data tab, or the Fuelbed Data tab. Choose Acquire Data from LANDFIRE.This will take you to the Acquire Data from LANDFIRE page.

On the Acquire Data from LANDFIRE page

  1. The Acquire Data from LANDFIRE page is where you name your data set and select an area of interest for your data using the map window. To set your area of interest, use one of these two options:
    1. Select Draw Box (a radio button along the top left of the map window) to manually draw your area of interest on the map. When you use this option, IFTDSS automatically enters the geographic coordinates in the coordinate entry boxes (the North, East, South, and West fields in the center top of the page).
    2. Alternatively, enter the geographic coordinate extents of your area of interest using the coordinate entry boxes.

  2. Once you have defined your area of interest, choose Next. LANDFIRE data for that area is retrieved.

       Note   

    The area of interest for your data set will automatically set the area of interest for a project when you choose to use the data set for a new project.

  3. After your data set has been successfully acquired, you are returned to the Saved Data Sets page, where a message confirms that your data was successfully acquired. Your new data set is in the list of available data sets. Because you created the data set directly and not as part of a project, there is no project associated with the data set yet.


    As with any other data set, you can edit, delete, rename, or copy the data set.

  4. To use this data set for a project, choose the Projects tab (at the top of the page) and select an existing project, or choose Home > Create a New Project.

  5. From the Projects tab, choose Create New Project (the link at the upper right of the page). The Create New Project page appears, where you can give your project a name and enter optional information. (For more information on creating a project, see Creating a Project.)
