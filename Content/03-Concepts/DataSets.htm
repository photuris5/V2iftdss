Data Sets and Data Management

One of the key features of IFTDSS is the ability to acquire and edit data for use in fuel treatment analysis and planning. The modules and tools available in IFTDSS rely on vegetation data from one of these sources:

  • manually selected fire behavior fuel models (the Anderson 13; the Scott and Burgan 40)
  • LANDFIRE
  • FCCS fuelbeds

To manage your data sets, choose Manage My Data Sets on the Home page, or access the Data tab. All saved data sets are displayed on this tab.

From the Data tab, you can do the following:

 

  • Acquire LANDFIRE data for a specified area of interest.
  • Upload new data sets from your local machine. IFTDSS supports uploading landscape data files (.lcp and .prj file formats) and shapefiles and FCCS fuelbed files (.xml format).
  • View each data set's status.
  • Edit, delete, rename, or copy existing data sets. If you have spatial data, such as LANDFIRE gridded data, you can edit the data by choosing the Edit link next to the data file. This launches the Data Studio, which provides tools for viewing and editing spatial data.
  • Sort data sets by name, project, creation date, or status.
  • Search for data sets. As you type in the Search field, the search results are displayed instantly.
  • Filter the view of data sets by project name. (Select the project name in the Filters drop-down box, and only data sets associated with the selected project are displayed.)
  • Select how many data sets you wish to view per page (using the Show drop-down box).

Supported Vegetation Data Sets

IFTDSS supports the following types of vegetation data sets:

  • LANDFIRE landscape files (.lcp) and accompanying geographic projection file (.prj)
  • FCCS Fuelbed files (.xml)
    •    Note   

    The standard FCCS Fuelbed files are selected when using the surface fire behavior for FCCS fuelbeds module or the natural fuels consumption module. Future versions of IFTDSS will support the use of custom FCCS fuelbed data.

There are two options for loading .lcp files into IFTDSS:

  1. You can upload a landscape (.lcp) file and corresponding projection (.prj) file from your personal data files.
  2. You can select an area of interest using map tools in IFTDSS. The corresponding .lcp data will be retrieved and loaded into IFTDSS.
