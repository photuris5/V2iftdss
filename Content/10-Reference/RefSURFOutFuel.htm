Surface Fire Behavior Outputs – Fuel

You can select one or more of these options.

Name Units Brief Description
Dead fuel load portion Percentage The percentage of the total fuel load that is dead fuel. The percentage is calculated after the load transfer is completed for dynamic fuel models. This variable is most meaningful in an examination of dynamic fuel models.
Dead herbaceous fuel load Tons per acre, tonne per hectare (Mg/ha) The weight of dead herbaceous fuel per unit area. This weight is a meaningful variable only for dynamic standard and custom fuel models. The initial value is zero. A portion of the live herbaceous fuel load is transferred into the dead herbaceous fuel load category. The portion of the load transferred is either specified by the user or calculated from live herbaceous fuel moisture content. The moisture content for this fuel category is the value assigned to the 1-hour fuel. The SA/V ratio is specified for the live herbaceous fuel.
Fuel load transfer portion Percentage The percentage of live herbaceous load transferred to the dead class. This variable is meaningful only for dynamic standard or custom fuel models. The load transfer portion can either be calculated from live fuel moisture as described by Scott and Burgan (2005), or it can be entered directly.
Live fuel load portion Percentage The percentage of the total fuel load that is live fuel. The percentage is calculated after the load transfer is completed for dynamic fuel models. This variable is most meaningful in an examination of dynamic fuel models.
Live herbaceous fuel load Tons per acre, tonne per hectare (Mg/ha) The weight of live herbaceous fuel that remains after some of the load is transferred to the dead herbaceous fuel class. This is a meaningful variable only for dynamic standard and custom fuel models.
Total dead fuel load Tons per acre, tonne per hectare (Mg/ha) The sum of the load of all dead fuel classes. For standard and custom fuel models, this sum includes 1-hour, 10-hour, 100-hour, and dead herbaceous fuel loads. The sum is calculated after the load transfer is completed for dynamic fuel models. This variable is most meaningful in an examination of dynamic fuel models.
Total live fuel load Tons per acre, tonne per hectare (Mg/ha) The sum of the load of all live fuel classes. For standard and custom fuel models, this sum includes live woody and live herbaceous fuel loads. The sum is calculated after the load transfer is completed for dynamic fuel models. This variable is most meaningful in an examination of dynamic fuel models.
