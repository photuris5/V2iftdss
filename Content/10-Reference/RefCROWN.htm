Crown Fire Behavior (IFT-crown)

You can use the IFTDSS crown fire behavior module to calculate the surface fire intensity that is needed for a surface fire to transition to a crown fire. The crown fire module implemented in IFTDSS produces the same results (which may differ slightly due to rounding) as the crown fire module in BehavePlus.

  • This module uses flame length or fireline intensity as input options, along with a number of input parameters.
  • The module provides separate sets of information on spread and intensity as output values.
