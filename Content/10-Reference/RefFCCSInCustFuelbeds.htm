Surface Fire Behavior (FCCS) Inputs – Customize Fuelbeds

On the Customize Fuelbed page, you enter fuelbed descriptions and source information for custom fuelbeds. Because customized fuelbeds are created by modifying fuelbed prototypes, you must replace the existing description with one that fits the custom fuelbed.

Name Units Brief Description
Fuelbed name NA A name for the fuelbed. You assign this name. The FCCS does not require unique names or any particular naming protocol for user fuelbeds.
Description NA A description of the fuelbed. You create and enter the description. Descriptions for the FCCS national fuelbeds generally include geographic representation, vegetation, and disturbance and/or management history.
Ecoregion(s) A drop-down list The list of of Bailey's 13 ecoregions. Site-specific fuelbeds generally only have one ecoregion, whereas modal fuelbeds are likely to span multiple ecoregions. The 13 ecoregions in the drop-down list are
  • Hot Continental
  • Marine
  • Mediterranean
  • Prairie
  • Rainforest
  • Savanna
  • Subarctic
  • Subtropical
  • Temperate Desert
  • Temperate Steppe
  • Tropical/Subtropical Desert
  • Tropical/Subtropical Steppe Desert
  • Tundra
  • Warm Continental

A map of these divisions is provided on the Fuelbed Selection page.
Vegetation form A drop-down list You can select one of seven vegetation forms from a drop-down list. With the exception of the "mixed forest" category, these vegetation forms are self-explanatory. Mixed forests are defined as forests in which evergreen and deciduous species each generally contribute 25% to 75% of total tree cover in both the overstory and midstory strata. The seven vegetation forms are
  • Broadleaf forest
  • Conifer forest
  • Grassland
  • Mixed forest
  • Savanna
  • Shrubland
  • Slash
.
Cover type A drop-down list Forest and/or rangeland cover types. Multiple cover types may apply to an individual fuelbed.
Change agent(s) A drop-down list A list of natural or management-induced activities that have altered a fuelbed. Multiple change agents may apply.
Structural class A drop-down list Structural class applies to forested fuelbeds only. These vegetation forms describe the number of canopy layers, relative size of trees, stage of understory development, and relative degree of stand closure. The structural class applies only to forested fuelbeds.

The options are
  • Old-forest multistory
  • Old-forest single story
  • Stem exclusion closed canopy
  • Stem exclusion open canopy
  • Stand initiation
  • Understory re-initiation
  • Young-forest multistory
  • Not used or Unknown

Graphical depictions of forest structural classes are provided on the Fuelbed Selection page.
Natural fire regime A drop-down list Select one of seven fire regime classifications:
  • RN: no natural fire (or very little)
  • R1: infrequent light surface fire (more than 25-year intervals)
  • R2: frequent light surface fires (1-25 year return intervals)
  • R3: infrequent, severe surface fires (more than 25-year return intervals)
  • R4: short return interval crown fires (25-100 year return intervals)
  • R5: long return interval crown fires and severe surface fires in combination (100-300 year return intervals)
  • R6: very long return interval crown fires and severe surface fires in combination (over 300-year return intervals)
Condition class A drop-down list Select one of three qualitative measures describing the degree of departure from historical fire regimes:
  • Class 1: fire frequencies have departed from historical frequencies by no more than one return interval and fire regimes are moderately altered.
  • Class 2: fire frequencies have departed from historical frequencies by more than one return interval and fire regimes are significantly altered.
  • Class 3: fire frequencies have departed from historical frequencies by multiple return intervals with dramatic changes to fire regimes.
Data quality rating A drop-down list FCCS National Fuelbeds contain an overall rating for the quality of the data used for creating the fuelbed. The five fuel data quality rankings are
  1. No data: no actual numbers for any data value; created from experience and/or ecological literature
  2. Partial data: less than 35% of modal data from literature, photo series, or peer-reviewed data source
  3. Partial data: 35-85% of modal data from literature, photo series, or peer-reviewed data source
  4. Data driven: 85-100% of modal data from photo series or peer-reviewed data source
  5. Data driven: 85-100% of all data (minimum, maximum, mode) from photo series or peer-reviewed data source

The default fuel data quality ranking for user fuelbeds is zero.
Photo series ID ID number A reference for the particular photo series that was used as source data to create some or all of the custom fuelbed. This feature is not implemented in the FCCS national fuelbeds but can be entered in customized fuelbeds.
User info section Seven optional entries In the user information section, you can record information that might be useful for your own file management. You can also provide contact information if you wish to submit fuelbeds to the FCCS administrator for consideration as additional prototypes. The optional user information entries are Date, Name, Email, Phone, Affiliation, Address, and Notes.
