Surface Fire Behavior (FCCS) Inputs – Litter, Lichen, and Moss

There are three forest layers to this type of FCCS input:

  1. Litter: The top layer of the forest or rangeland floor; composed of loose debris of dead sticks, branches, twigs, dead grass, and recently fallen leaves or needles that have been minimally altered by decomposition.
  2. Lichen: Ground lichens occurring on rocks, bare ground, or low vegetation.
  3. Moss: Low-growing bryophytes, which usually occur in moist habitats.

 

Name Units Brief Description
Depth Inches Total depth of litter, lichen, or moss layer.

Min: minimum depth.
Mode: most common depth.
Max: maximum depth.
Litter arrangement A drop-down list Specifies the spatial arrangement of litter, which can be one of the following choices:
  • Fluffy (freshly fallen)
  • Normal
  • Perched (on grass or forb)
Litter type A drop-down list Specifies the type of litter, which can be one of the following choices:
  • short needle pine
  • long needle pine
  • other conifer
  • broadleaf deciduous
  • broadleaf evergreen
  • palm fronds
  • grass
Moss type A drop-down list Specifies the type of moss, which can be one of the following choices:
  • sphagnum moss
  • other moss (e.g., feathermoss)
Percent cover Percentage Total percentage cover of litter, lichen, or moss layer.

Min: minimum percentage cover.
Mode: most common percentage cover.
Max: maximum percentage cover.
Relative cover Percentage Relative percentage cover of each litter type. Must sum to 100%.

Min: minimum percentage cover.
Mode: most common percentage cover.
Max: maximum percentage cover.
