Crown Scorch Height (IFT-scorch)

You can use the crown scorch height model (IFT-scorch) to estimate the crown scorch height from surface fire flame lengths. The crown scorch height model implemented in IFTDSS produces the same results (which may differ slightly due to rounding) as the scorch height module in BehavePlus.

  • Input variables include wind speed, flame length, and air temperature.
  • Scorch height is provided as an output.
   Note   

The crown scorch height model can produce scorch heights greater than the tallest tree (379.1 ft). Using the maximum range for all inputs, this model will predict a crown scorch height of 20,541.61 ft.
