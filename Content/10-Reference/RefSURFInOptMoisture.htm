Surface Fire Behavior Input Options – Moisture

Name Units Brief Description
Dynamic curing load transfer NA You have two options:
  • is calculated from live herbaceous fuel moisture.

    With this option, dynamic fuel models include the transfer of live herbaceous fuel load to the dead category as a function of live herbaceous fuel moisture (Scott and Burgan, 2005). Seventeen of the 40 new fuel models are dynamic.
  • is input directly.

    With this option, rather than calculating the load transfer portion from live fuel moisture, you can enter the dynamic curing load transfer directly. With this option, you can examine the influence of the load transfer function on calculated fire behavior (Jolly, 2007; Andrews et al., 2006).
Moisture is entered NA You have three options:
  • by dead and live category.

    With this option, the moisture content value entered (as a percentage) for "dead" is used for 1-hour, 10-hour, and 100-hour fuel. The moisture content for "live" is used for live woody and live herbaceous fuel. This value does not include foliar moisture (conifer needles).
    •
  • by individual size class.

    With this option, the moisture content is entered (as a percentage) for each fuel size class, including 1-hour, 10-hour, and 100-hour dead fuels and herbaceous and woody live fuels.
  • by moisture scenario.

    (For a list of the scenarios, see FCCS Inputs – Environmental Parameters: Standard Fuel Moisture Scenarios.) With this option, you select a moisture scenario from a drop-down list. A moisture scenario is a predefined set of fuel moistures for 1-hour, 10-hour, and 100-hour dead fuel and for herbaceous and woody live fuel. A moisture scenario is analogous to the fuel model concept in that a single code represents a set of fuel. Fuel moisture scenarios may be developed, for example, to represent local 90-, 95-, and 97-percentile weather situations.
