Consumption, Emissions, and Tree Mortality (IFT-FOFEM)

The consumption, emissions, and tree mortality modules can be used to predict fuel consumption, emissions, tree mortality, and smoke production for individual stands.

  • Fuel loading and moisture information must be provided as input.
  • Outputs include consumption by fuel type and emissions by air pollutant.

Consumption and Emissions

You can use the consumption and emissions module to predict fuel consumption, emissions, and smoke production caused by a prescribed fire or wildfire.

  • Input variables include forest floor characteristics, fuel loading and moisture values, and percentage of crown burn.
  • Output variables include the amount of fuel consumed by size categories and the amount of emissions generated from flaming and smoldering fires.

Tree Mortality

You can use the tree mortality module to predict tree mortality caused by a prescribed fire or wildfire.

  • Input variables include tree characteristics (e.g., species, DBH, height, and crown ratio) and fire characteristics (e.g., flame length and scorch length).
  • Output variables include post-fire stand density, number of trees killed by fire, and various other tree mortality statistics.
