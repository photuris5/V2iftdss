Surface Fire Behavior Outputs – Slope

You can select one or more of these options.

Name Units Brief Description
Slope elevation change Feet, meters The difference in elevation between the two points used to find map distance. This parameter is an intermediate value in the calculation of slope steepness from map measurements.
Slope horizontal distance Feet, meters The ground distance between two points. This parameter is an intermediate value in the calculation of slope steepness from map measurements.
Slope steepness Degrees or percentage The maximum steepness of the slope. It is expressed as either degrees or percentage depending on the input option that is selected. This value is not necessarily the slope steepness in the direction of spread.
