Burn Probability from Random Ignition (IFT-FlamMap-RANDIG)

RANDIG is a mechanistic fire spread program that uses the minimum travel time algorithms in FlamMap to simulate fire spread across landscapes using constant wind and fuel moisture inputs (Finney, 2006).

In IFTDSS, the burn probability module (IFT-FlamMap-RANDIG) is used to simulate burn probability across an entire landscape. The burn probability module creates raster maps of composite burn probabilities (burn probabilities for all flame lengths) and raster maps of burn probabilities for a set of user-defined flame length classes. This module can be used for hazard analysis and is an essential component of the risk assessment workflow.

Supporting information is provided in the response functions table and in the flame length interpretations table.
