Surface Fire Behavior (FCCS) Inputs – Woody Fuels: Rotten Wood

The Rotten Wood category displays loading (tons/acre) data for a range of size classes. Species designations and relative cover apply to all rotten wood size classes.

Name Units Brief Description
Greater than twenty inches loading Tons per acre Loading of huge (> 20 inch) rotten wood.

Min: minimum loading.
Mode: most common loading.
Max: maximum loading.
Nine to twenty inches loading Tons per acre Loading of very large (> 9 to ≤ 20 inch) rotten wood.

Min: minimum loading.
Mode: most common loading.
Max: maximum loading.
Relative cover Percentage
 Relative percentage cover of species in all rotten wood categories. Must sum to 100%.

Min: minimum percentage cover.
Mode: most common percentage cover.
Max: maximum percentage cover.
Scientific name A drop-down list Scientific (Latin) name of species in all rotten wood categories. You can select a maximum of two species.
