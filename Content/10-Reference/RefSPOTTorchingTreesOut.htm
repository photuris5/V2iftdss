Spotting Distance Output Parameters – Torching Trees

Name Units Brief Description
Cover Height Applied for Spotting Distance from Torching Trees Feet, meters The cover height actually used in the computation of spotting distance. This is an intermediate value in the calculations that might be of interest to users examining the spotting distance module. A critical cover height is first determined from the maximum firebrand height. The applied cover height is the greater of the critical cover height and the input canopy height.
Firebrand Height from Torching Trees Feet, meters The maximum height to which a firebrand can be lofted and still be capable of starting a fire when it lands.
Flat Terrain Spotting Distance from Torching Trees Miles, kilometers The maximum spotting distance over flat terrain. This value is then adjusted according to the terrain descriptors.
Spot Distance from Torching Trees Miles, kilometers The maximum distance at which one can expect potential spot fires resulting from firebrands from torching trees. The spotting module is applicable under conditions of intermediate fire severity in which spotting distances up to a mile or two might be expected. This module does not apply to those extreme situations where spotting may occur up to tens of miles from the fire front.
Steady State Flame Duration from Torching Trees NA A dimensionless value (unitless) used to calculate firebrand height.
Steady State Flame Height from Torching Trees Feet, meters Used to calculate the height to which a firebrand is lofted.
Tree Height/Flame Height Ratio for Torching Trees A ratio The ratio of the torching tree height to the steady state flame height is used to determine the appropriate equation for firebrand height.
