Creating a Project

Because you use projects to organize your data, one of the first things you might want to do is create a project. A project is like a workspace. A project will be assigned a geographic area of interest. Once a project is created, you may start performing modeling runs within it.

To create a project,

  1. Choose Create New Project.

Alternatively, you may click on the Projects tab of IFTDSS and select Create a New Project at the top right of the page.

  1. This opens the Create New Project page, where you enter the details for your project. Enter a descriptive project name, which is required. Everything else is optional, though the more information you include now, the better. For this tutorial we'll name the Project 'Squaw Creek'.

  1. Choose Next. By default you will be taken to the Runs page. Click the name of the project you just created at the top left of the page. This will take you to the Project Summary page.

From the Project Summary page, go to the Area of Interest section and begin the process of acquiring data by selecting the Acquiring data from LANDFIRE option.

On the next page you will be prompted to select the source of your data. For this tutorial, select Acquire data from LANDFIRE. For future projects, there are also choices for using existing datasets, or uploading your own dataset.

Click Next to continue.
