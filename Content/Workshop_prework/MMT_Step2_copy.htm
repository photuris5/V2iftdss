Creating a New Run

From the IFTDSS Dashboard, go to the project summary page for the project in which you are working. This is done via the Projects tab, or the Manage My Projects link.

From the Project Summary Page select Create a New Run, and choose the type of run you would like to initiate. You may access MTT through either the Hazard Analysis or Fuels Treatment pathways. The hazard analysis pathway allows a single MTT run while the Fuels Treatment pathway allows the user to run both a pre and post-treatment analysis within the same run that can be compared on a single map page. For this tutorial, we will choose the Fuels Treatment pathway.

Choosing either pathway will require clicking through several steps. For this example, choose the Fuels Treatment pathway>Fuels Treatment across the Landscape>LANDFIRE and then select the Manual treatment location (user-defined treatments) (IFT-MTT) model run.

On the Set up page, you will name the run, and have the opportunity to change the extent. By default, runs will use the same extent as the project area of interest. However, you may make runs that are smaller than the project area of interest (but not larger). For this tutorial, we will leave the extent at its default setting.

Give the run a descriptive name, such as 'Squaw Crk. Rx Fire Landscape' and click Next.
