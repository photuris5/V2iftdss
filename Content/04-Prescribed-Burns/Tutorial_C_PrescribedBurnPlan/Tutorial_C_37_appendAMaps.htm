Appendix A: Maps

Map Requirements

At a minimum, the plan will include a vicinity and project map.

The number of maps, map size and scale, legend, and level of detail should be appropriate for the complexity of the project.

All maps will include the following standard map elements

  • Title
  • Name of Preparer(s)
  • Date
  • North Arrow
  • Scale
  • Legend

(USDA and U.S. DOI, 2014, p. 27)

Vicinity Map

Map scale will be such that the burn units can be located on the ground and in sufficient detail to guide implementation.

Project Map

Identifies features in sufficient detail to guide and assist in operational implementation of the prescribed fire.
