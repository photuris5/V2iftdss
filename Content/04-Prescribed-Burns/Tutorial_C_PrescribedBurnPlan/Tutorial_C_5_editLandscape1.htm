Editing Landscape Data (part 1)

After acquiring the LANDFIRE data, you are returned to the Project Summary page. We will now review and edit the landscape data for the project area.

At the bottom of the Project Summary page, under the Project Data section, click on the Action Icon, and select Edit. The Data Studio window appears.

   Note   

Pop-up blockers must be disabled in order to open Data Studio.
