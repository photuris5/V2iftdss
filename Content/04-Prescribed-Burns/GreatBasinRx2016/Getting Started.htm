Getting Started

To begin, log into IFTDSS and click the Create a New Project option under the Home tab.

  • Choose a descriptive project name.
  • If desired, fill in the optional information.
  • Choose Next

After creating a new project, you will see the page for choose the type of run you want to create.

Choose the type of run you would like to create by choosing the following links: Prescribed Burn Planning ===> Create a burn plan document

Name your run and choose Next.
