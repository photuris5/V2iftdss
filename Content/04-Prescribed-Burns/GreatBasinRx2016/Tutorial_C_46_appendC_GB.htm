Appendix C: Complexity Analysis

Once you generate your burn plan you will have access to Appendix C: Complexity Analysis. Using this worksheet and the descriptors from the Prescribed Fire Complexity Rating System Guide you will be able to populate Element 3. The complexity analysis contains 14 elements with three factors to consider for each element:

In this step you will fill out the complexity analysis and summarize findings at the end. The final ratings populate Element 3.

  • Factor 1 is Risk: the probability an adverse event will occur.
  • Factor 2 is Potential Consequences: the measure of cost/result of an adverse event.
  • Factor 3 is Technical Difficulty: the skill needed to implement the burn and deal with potential adverse events.

Values are assigned for each of the factors: low, moderate, or high.

Refer to the Prescribed Fire Complexity Rating System Guide for more details.
