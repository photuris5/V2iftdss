Prescribed Burn Plan Tutorials

To illustrate how to use the prescribed burn workflow in IFTDSS to complete a burn plan, we have prepared some tutorials that walk through some workflows in IFTDSS.

Assessing Fire Hazard for Prescribed Burn Planning walks through how to use the hazard analysis workflow in IFTDSS to identify areas that warrant further analysis based on potential fire hazard. The tutorial covers:

  • Acquiring LANDFIRE data and setting up a project in IFTDSS
  • Creating a run for fire behavior across a landscape (IFT-FlamMap)
  • Examining data using Google Earth
  • Identifying potential fire hazards outside of a burn unit to help develop holding and contingency plans

Preparing a Prescribed Burn Plan walks through the preparation of a burn plan document:

  • Setting up a project in IFTDSS
  • Creating and filling in the burn plan template
  • Establishing burn objectives and environmental thresholds
  • Using the Prescribed Burn Planning Workflow
  • Generating and saving data and maps to include in the burn plan document
  • Generating the burn plan as a Microsoft Word document and customizing the plan
