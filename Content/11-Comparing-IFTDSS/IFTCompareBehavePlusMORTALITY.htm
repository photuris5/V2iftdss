BehavePlus MORTALITY Compared to “Tree Mortality” as Implemented in IFTDSS

The BehavePlus 5.0.5 MORTALITY module is not used to calculate tree mortality in IFTDSS. Instead, IFTDSS includes a set of FOFEM 4.0 algorithms and options for calculating tree mortality.
