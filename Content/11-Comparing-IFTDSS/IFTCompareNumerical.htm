Numerical Comparison of Surface Fire Behavior Modules

As scientific models are implemented in IFTDSS modules, each module is tested to ensure that it is

  • functioning properly within the software framework (software testing).
  • producing the correct numeric outputs (scientific testing). The scientific test procedures include comparing the outputs as implemented in IFTDSS to the outputs from their native desktop application(s).

The following two fire behavior modules in IFTDSS were compared with the fire behavior modules implemented in the desktop applications of BehavePlus 5.0.0 and NEXUS 2.0 for select fuel types, fuel moisture scenarios, topography, and winds:

  • Surface fire behavior for individual stands (IFT-FlamMap): using the FlamMap 3.0 dynamic link libraries (.dlls).
  • Surface fire behavior for individual stands (IFT-surface): using the fire behavior software developer kit (FBSDK) algorithms and software code and a set of BehavePlus code libraries.

Summary of Findings

The fire behavior module outputs from IFTDSS compared well with the fire behavior module outputs from BehavePlus 5.0.5 and NEXUS 2.0. Small differences in output values were observed; however, they can be attributed to rounding and truncating among the different modules. However, these differences are not large enough to alter the interpretation of the module results.
