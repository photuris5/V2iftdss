BehavePlus CONTAIN Compared to IFT-contain

In the IFT-contain module, IFTDSS provides all the BehavePlus input functionality to estimate escaped fire containment success for the suppression efforts of a single resource.

IFTDSS also provides all the BehavePlus outputs for a single resource except the BehavePlus fire containment diagram.

IFTDSS does not include the multiple resource options found in BehavePlus. Moreover, the IFTDSS fire behavior modules are not linked to the IFTDSS containment resources module.
