Getting Started (Instructions, Tutorials, and Videos)

Welcome to IFTDSS, the Interagency Fuels Treatment Decision Support System. IFTDSS provides data and tools to support fuels treatment analysis and planning. IFTDSS also includes tools to use when developing burn plans and burn plan documentation, and when performing a risk assessment.

In addition to the links on this page, see the following topics for more information and additional tutorials:

  • Hazard analysis tutorials
  • Prescribed burn planning tutorials
  • Risk assessment tutorials
  • Fuels treatment tutorials
