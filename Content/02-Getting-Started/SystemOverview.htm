System Overview

The following image shows the layout of the navigation portion of the IFTDSS user interface:

Navigation tabs appear along the top of the screen. These tabs provide access to general areas within IFTDSS and appear on every page.

Your user image and profile information appear on the right side of the Home page.  From here you can manage and edit your profile information.

You can navigate through the system using the links on the home page and/or the navigation tabs along the top of the page. When you are working with projects and runs, breadcrumbs (which are links back to each previous page) appear along the top left area of the screen. You can use these breadcrumbs to navigate within project and run screens.

On the left side of the Home page, there is a menu of options. Each of these is described in the following text, with links to pages with more details.

Create a New Project

Organize your work by creating a project. In IFTDSS, a project is like a workspace. A single project can contain multiple model runs and analysis activities. For information on creating a new project, see Creating a Project.

Manage My Projects

Once you have created one or more projects, you can access and manage those projects on the Active Projects page. There are two ways to access that page: the Manage My Projects link or the Projects tab.

Choosing either of these takes you to the Active Projects page. There, you can view and manage existing projects. You can also create a new project by accessing the Create New Project link on this page.

Manage My Data Sets

Use this menu option (or click directly on the Data tab) to access the page where you can upload and manage project data.

Search Published Projects

In future versions of IFTDSS, you will be able to publish and share projects with other IFTDSS users.

Find Other Users

Use this menu option (or access the Collaborate tab) to find other IFTDSS users.

Instructions, Tutorials, and Videos

This online help comes with a variety of resources providing information on using IFTDSS effectively. The Getting Started page summarizes the tutorial material in the online help, providing links as needed.

Use this menu option to learn what's new in this version of IFTDSS.
