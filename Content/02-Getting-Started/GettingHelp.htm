Getting Help

You can access the online help and documentation (which you are viewing right now) to aid your understanding of how to use IFTDSS, explore fuel and fire models, and find the definitions of fire model input and output parameters. The help also describes available features within IFTDSS. Help is available on nearly every page in IFTDSS by selecting the help icon.

We welcome your comments, suggestions, and other feedback.

Citations and Bibliography

Where appropriate, citations are given in this documentation using standard citation forms; for example, (Author, Year). The first time a citation appears on a page, it will also have a link to the full reference in the bibliography; for example, (Rothermel, 1991). In the bibliography, with only a few exceptions, there are links to the full publication on the Internet. By default, these links will open inside the help frame.
